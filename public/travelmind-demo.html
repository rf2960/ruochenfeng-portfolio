Agentic AI / product walkthrough

TravelMind planner demo.

A portfolio walkthrough for a multi-agent LLM travel-planning concept. The project demonstrates destination discovery, existing-plan refinement, structured itinerary generation, and product-oriented system design.

System concept

What the demo shows

01

Constraint parsing

User intent, dates, budget, pace, and preferences are normalized before planning.

02

Agent routing

Discovery and plan-enhancement flows are separated so each mode can use specialized logic.

03

Structured output

The product returns readable itineraries, warnings, assumptions, scores, and next-step refinements.

Product screens

Two user flows.

Discovery input

Users enter flexible trip preferences when they need destination recommendations.

Discovery result

The planner returns ranked options with structured trip reasoning and practical guidance.

Plan enhancement input

Users can paste an existing rough plan and ask for improvements.

Enhanced itinerary

The output improves pacing, detail, warnings, and itinerary structure.
